# UI UX Pro Max

An AI skill that provides design intelligence for building professional UI/UX across multiple platforms and frameworks.

## Core Design Principles

### 1. Pattern: Hero-Centric + Social Proof
- **Conversion**: Emotion-driven with trust elements.
- **CTA**: Above fold, repeated after testimonials.
- **Sections**: 1. Hero, 2. Services, 3. Testimonials, 4. Booking, 5. Contact.

### 2. Style: Soft UI Evolution
- **Keywords**: Soft shadows, subtle depth, calming, premium feel, organic shapes.
- **Performance**: Excellent | Accessibility: WCAG AA.

### 3. Typography Strategy
- **Mood**: Elegant, calming, sophisticated.
- **Pairing**: Serif for headings (Sophistication) / Sans-serif for body (Readability).
- **Rounded Fonts**: Best for "Friendly" and "Modern" feel.

### 4. Key Effects
- **Transitions**: Smooth 200-300ms transitions.
- **Hover States**: Gentle but clear feedback.
- **Interactions**: Magnetic effects, parallax, and scroll-triggered animations.

## Pre-delivery Checklist
- [ ] No emojis as icons (use SVG: Lucide/Heroicons).
- [ ] `cursor-pointer` on all clickable elements.
- [ ] Hover states with smooth transitions (150-300ms).
- [ ] Responsive: 375px, 768px, 1024px, 1440px.
- [ ] Accessibility: Contrast 4.5:1 minimum.
- [ ] Focus states visible for keyboard navigation.
- [ ] `prefers-reduced-motion` respected.
